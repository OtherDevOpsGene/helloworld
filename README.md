# helloworld

The most basic of Java programs. A simple Java application that compiles to a
`.jar` file using Maven.

## Building

```bash
mvn clean package
```

## Running

```console
$ java -cp target/helloworld-2.0.jar dev.otherdevopsgene.helloworld.HelloWorld
Hello, world! The current time is 2:23:59 PM on November 26, 2022.
```
